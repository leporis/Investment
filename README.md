## 작업별 참조 지침

### 1. 잔고 관리 (Balance Management)

**잔고 엑셀 파일 수정 시 반드시 준수:**

- **주 지침**: `AccountManagementGuidelines` (계좌관리지침)
- **작업 템플릿**: `StockBalanceTaskPrompt` (작업지시 프롬프트)
- **파일명 규칙**: `DocumentVersionControlGuide` (문서버전관리가이드)

**적용 범위**: 매매이력 입력, 종가 업데이트, 손익요약 갱신, 실현손익차트 갱신, 월별 마감 처리

**핵심 규칙**:
- 잔고 파일명: `잔고_YYYYMMDD_HHMM.xlsx`
- 매매이력 추가 시 4개 시트(주식잔고, 매매이력, 손익요약, 실현손익차트) 동시 업데이트
- 0주 종목은 주식잔고 시트에서 행 삭제
- 모든 수치는 웹 검색으로 재확인 필수

---

### 2. 포트폴리오 운영 전략 (Portfolio Strategy)

**포트폴리오 운영 전략 문서 작성 시 반드시 준수:**

- **주 지침**: `PortfolioStrategyGuidelines` (포트폴리오운영전략작성지침)
- **파일명/이력**: `DocumentVersionControlGuide` (문서버전관리가이드)

**적용 범위**: 등급 분류(A~F), 추세 분석, 섹터별 전망 분석, 목표가/손절가 산정, 관심종목 분석, 종목별 상세 전략

**핵심 규칙**:
- 파일명: `포트폴리오운영전략_vX.Y.Z.W_키워드_YYYY-MM-DD_HHMM.md`
- 시황/뉴스 내용은 포함하지 않음 (별도 `시황과뉴스_YYYYMMDD_HHMM.md`로 분리)
- 목표가에 섹터전망계수(0.80~1.20) 반영
- 증권사 리포트 기반 목표가 산정

---

### 3. 문서 버전 관리 (Document Version Control)

**모든 문서 생성/수정 시 반드시 준수:**

- **주 지침**: `DocumentVersionControlGuide` (문서버전관리가이드)

**적용 범위**: 프로젝트 내 생성되는 모든 문서 (.md, .xlsx, .docx 등)

**핵심 규칙**:
- 버전 체계: `vX.Y.Z.W` (Major.Minor.Revision.Patch)
- 파일명: `주요내용_v버전_수정사항키워드_날짜_시간.확장자`
- 문서 끝에 **참고자료** + **생성/수정 이력** 필수 포함
- 세션 인계 시 인계 문서 생성
