## 2. 잔고 파일 관리

### 2.1 파일명 규칙

```
잔고_YYYYMMDD_HHMM.xlsx

예: 잔고_20260123_1430.xlsx
```

- 잔고 엑셀 파일 업데이트 시 파일명 자동 변경
- 날짜와 시간은 작업 완료 시점 기준

### 2.2 시트 구성 (4개)

| 시트명 | 용도 |
|--------|------|
| 주식잔고 | 보유 종목 현황 (평균매입가, 보유수량, 현재가, 평가손익) |
| 매매이력 | 거래 기록 (매수/매도 내역, 실현손익) |
| 손익요약 | 총 실현손익, 평가손익, 투자원금, 현재 평가액, 총 수익률 |
| 실현손익차트 | 차트용 데이터 (일자별 실현손익 기록) |

### 2.3 문서 업로드 시 확인사항

- 포트폴리오 관련 문서 업로드 시 최종 수정일이 현재 날짜보다 **1개월 이상** 지났으면 자동으로 알림

## 3. 매매 이력 업데이트

### 3.1 4개 시트 동시 업데이트 규칙

매매 이력 추가 시 잔고 파일의 **모든 시트**를 업데이트:

#### 1) 주식잔고 시트
- 보유현황 업데이트 (수량, 평균매입가 등)
- **0주 종목은 행 삭제**

#### 2) 매매이력 시트
- 거래기록 추가
- 날짜순 정렬
- 실현손익: 수익(+) 빨간색, 손실(-) 파란색

#### 3) 손익요약 시트
- 총 실현손익 업데이트
- 종목별 실현손익 집계

#### 4) 실현손익차트 시트
- 총 실현손익 행 **위에** 새 행 삽입
- 형식: 일자 | 종목명 | 실현손익
- 총 실현손익 행: A열(총실현손익) | B열(빈칸) | C열(합계, 굵은글씨)

## 4. 손익요약 업데이트

### 4.1 당일 종가 입력 시 업데이트 항목

| 항목 | 설명 |
|------|------|
| 평가손익 | 현재 보유 중인 종목의 미실현 손익 |
| 투자원금 | 총 매입금액 |
| 현재 평가액 | 현재가 × 보유수량 합계 |
| 총 수익률 | (현재 평가액 - 투자원금) / 투자원금 × 100 |

## 5. 포트폴리오 전략 문서

### 5.1 작성 지침

- **포트폴리오운영전략작성지침_v1.1.0.0** 준수
- 시황/뉴스 내용은 포함하지 않음 (별도 문서로 분리)

### 5.2 현재가 기준

| 시점 | 현재가 기준 |
|------|------------|
| 장 종료 후 | 당일 종가 |
| 장 종료 전 | 전일 종가 |

- **모든 종목 현재가는 웹 검색으로 재확인 필수**

---

## 6. 문서 작성 원칙

### 6.1 문서버전관리가이드 준수

모든 문서 생성/수정 시 **문서버전관리가이드_v1.2.2** 필수 준수:

- 버전 형식: vX.Y.Z.W
- 파일명: 주요내용_v버전_수정사항_날짜_시간.확장자
- 문서 끝에 **참고자료**와 **생성/수정 이력** 필수 포함

### 6.2 시황과 뉴스 문서

별도 문서로 작성:
- 파일명: `시황과뉴스_YYYYMMDD_HHMM.md`
- 포트폴리오운영전략 문서와 분리
