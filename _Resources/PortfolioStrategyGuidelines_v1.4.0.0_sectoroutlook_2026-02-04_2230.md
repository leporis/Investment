## 1. 문서 개요

### 1.1 목적

포트폴리오 운영 전략 문서의 일관성 있는 작성과 품질 관리를 위한 지침

### 1.2 적용 범위

- 포트폴리오 운영 전략 문서 (포트폴리오운영전략_vX.Y.Z.W_*.md)
- 시황과 뉴스는 별도 문서로 분리 (시황과뉴스_YYYYMMDD_HHMM.md)

### 1.3 현재가 기준

| 시점 | 현재가 기준 |
|------|------------|
| 장 종료 후 | 당일 종가 |
| 장 종료 전 | 전일 종가 |

⚠️ **모든 종목 현재가는 웹 검색으로 재확인 필수**

## 2. 등급 분류 기준

### 2.1 기본 등급 (수익률 기준)

| 등급 | 수익률 범위 | 설명 |
|------|-------------|------|
| **A** | +10% 이상 | 수익 확보 단계 |
| **B** | 0% ~ +10% | 본전 이상 |
| **C** | 0% ~ -20% | 경미한 손실 |
| **D** | -20% ~ -40% | 중대한 손실 |
| **F** | -40% 이하 | 심각한 손실 |

### 2.2 등급 조정 (추세 + 섹터 전망 반영)

추세 점수와 섹터 전망에 따라 기본 등급을 조정합니다.

| 추세 점수 | 등급 조정 |
|----------|----------|
| +2 이상 | 한 단계 상향 고려 (예: D → C) |
| +1 | 현 등급 유지, 긍정적 관찰 |
| 0 | 현 등급 유지 |
| -1 | 현 등급 유지, 주의 관찰 |
| -2 이하 | 한 단계 하향 고려 또는 우선 매도 대상 |

| 섹터 전망 | 등급 조정 |
|----------|----------|
| ⭐⭐⭐⭐⭐ 매우 긍정 | 목표가 +10~20% 상향, 보유 비중 확대 검토 |
| ⭐⭐⭐⭐ 긍정 | 목표가 +5~10% 상향 |
| ⭐⭐⭐ 중립 | 기존 목표가 유지 |
| ⭐⭐ 부정적 | 목표가 -5~10% 하향, 손절 시기 앞당김 |
| ⭐ 매우 부정적 | 목표가 -10~20% 하향, 즉시 손절 검토 |

## 3. 추세 분석 기준

### 3.1 추세 점수 산정

각 종목에 대해 4가지 지표로 추세 점수를 산정합니다.

| 지표 | 상승 추세 (+1) | 횡보 (0) | 하락 추세 (-1) |
|------|---------------|----------|---------------|
| **1개월 주가 변동** | +10% 이상 | ±5% 이내 | -10% 이상 |
| **20일 이평선 대비** | 현재가 > MA20 | ±3% 이내 | 현재가 < MA20 |
| **거래량 추이** | 증가 추세 | 보합 | 감소 추세 |
| **52주 고저 위치** | 상위 20% | 중간 60% | 하위 20% |

### 3.2 추세 점수 해석

| 총점 | 해석 | 전략 방향 |
|------|------|----------|
| +3 ~ +4 | 강한 상승 추세 | 보유 유지, 추가 매수 검토 |
| +1 ~ +2 | 약한 상승 추세 | 보유 유지, 목표가 대기 |
| 0 | 횡보 | 현 전략 유지 |
| -1 ~ -2 | 약한 하락 추세 | 손절 검토, 반등 모니터링 |
| -3 ~ -4 | 강한 하락 추세 | 즉시 손절 권장 |

## 4. 섹터별 전망 분석 (신규)

### 4.1 목적

개별 종목 분석에 앞서 해당 섹터의 중장기 전망을 파악하여 목표가 산정과 매매 전략에 반영합니다.

### 4.2 분석 대상 섹터

보유 종목 및 관심 종목이 속한 주요 섹터를 분석합니다.

| 섹터 | 세부 분류 | 대표 종목 예시 |
|------|----------|---------------|
| **반도체** | 메모리, 비메모리, 장비, 소재 | 삼성전자, SK하이닉스, 하나마이크론 |
| **2차전지/소재** | 배터리 셀, 양극재, 음극재, 전해질, 장비 | LG에너지솔루션, 에코프로비엠, 천보 |
| **바이오/헬스케어** | 신약, CDMO, 의료기기, 진단 | 삼성바이오로직스, 셀트리온, 루닛 |
| **방산/우주항공** | 무기, 항공, 우주 | 한화에어로스페이스, 풍산, 나라스페이스 |
| **AI/IT** | AI 소프트웨어, 데이터센터, 플랫폼 | 네이버, 카카오, 솔트룩스 |
| **자동차/전기차** | 완성차, 부품 | 현대차, 기아, 삼성SDI |
| **화학/소재** | 정밀화학, 기초소재 | 코오롱인더, 롯데케미칼 |

### 4.3 전망 평가 기준

각 섹터에 대해 5단계 전망 등급을 부여합니다.

| 등급 | 전망 | 판단 기준 |
|:----:|------|----------|
| ⭐⭐⭐⭐⭐ | 매우 긍정 | 시장 성장률 +20% 이상, 슈퍼사이클 진입, 정책 수혜 |
| ⭐⭐⭐⭐ | 긍정 | 시장 성장률 +10~20%, 업황 회복 |
| ⭐⭐⭐ | 중립 | 시장 성장률 ±10% 이내, 선별적 접근 필요 |
| ⭐⭐ | 부정적 | 시장 성장률 -10% 이상, 업황 악화 |
| ⭐ | 매우 부정적 | 구조적 하락, 산업 쇠퇴 |

### 4.4 전망 분석 자료원

섹터 전망 분석 시 다음 자료를 우선적으로 활용합니다.

| 우선순위 | 자료원 | 활용 방법 |
|:--------:|--------|----------|
| 1 | **증권사 섹터 리포트** | 목표가 컨센서스, 업황 전망 |

| 4 | **기업 IR 자료** | 가이던스, 수주 현황 |
| 5 | **뉴스/언론** | 최신 동향, 이슈 |

#### 4.5.1 목표가 조정

## 5. 목표가/손절가 산정

### 5.1 목표가 산정 기준

**수익 종목 (A/B등급):**

| 구분 | 산정 방법 |
|------|----------|
| 1차 목표가 | MIN(증권사 컨센서스, 52주 고점 × 0.95) × 섹터전망계수 |
| 2차 목표가 | MAX(저항가격, 52주 고점) × 섹터전망계수 |
| 장기 보유 | 20% 물량은 추가 상승 기대 |

**섹터전망계수:**
- ⭐⭐⭐⭐⭐: 1.15 ~ 1.20
- ⭐⭐⭐⭐: 1.05 ~ 1.10
- ⭐⭐⭐: 1.00
- ⭐⭐: 0.90 ~ 0.95
- ⭐: 0.80 ~ 0.85

**손실 종목 (C/D등급):**

| 구분 | 산정 방법 |
|------|----------|
| 회복 목표가 | MIN(매입가 × 1.05, 저항가격 × 0.9) |
| 부분 회복 | 매입가 대비 -5% ~ -10% 수준 |

### 5.2 손절가 산정 기준

| 등급 | 손절가 기준 |
|------|------------|
| A | 현재가 대비 -7% (수익 보호) |
| B | 현재가 대비 -10% (본전 방어) |
| C | 매입가 대비 -15% 추가 하락 시 |
| D | 즉시 손절 또는 2주 유예 후 결정 |
| F | **즉시 손절 필수** |

### 5.3 증권사 리포트 활용

목표가 산정 시 증권사 리포트를 적극 활용합니다.

| 활용 기준 | 내용 |
|----------|------|
| **컨센서스** | 3개 이상 증권사 목표가 평균 활용 |
| **최신 리포트** | 1개월 이내 발행 리포트 우선 |
| **투자의견** | 매수 → 긍정, 중립 → 주의, 매도 → 손절 검토 |
| **기관 수급** | 외국인/기관 순매수 시 목표가 상향 검토 |

### 5.4 산정 근거 명시

각 목표가/손절가에 대해 반드시 산정 근거를 명시합니다.

```markdown
| 구분 | 가격 | 산정 근거 |
|------|------|-----------|
| 목표가 | 45,000원 | • 증권사 평균 목표가 42,000원<br>• 섹터 전망 ⭐⭐⭐⭐⭐ (×1.07)<br>• 52주 고점 43,250원 대비 +4% |
| 손절가 | 32,000원 | • 현재가 대비 -11%<br>• 지지선 33,000원 하단 |
```

## 6. 관심종목 분석

### 6.1 관심종목 분류

| 등급 | 의미 | 기준 |
|:----:|------|------|
| ★★★ | 즉시 매수 후보 | 추세 +2 이상, 섹터 전망 ⭐⭐⭐⭐ 이상 |
| ★★ | 단기 관찰 후보 | 추세 0~+1, 섹터 전망 ⭐⭐⭐ 이상 |
| ★ | 장기 관찰 대상 | 잠재력 있으나 당장 매수 부적합 |

### 6.2 선정 기준

| 구분 | 필수 조건 | 가점 조건 |
|------|----------|----------|
| 섹터 | 섹터 전망 ⭐⭐⭐ 이상 | 섹터 전망 ⭐⭐⭐⭐⭐ |
| 시가총액 | 500억원 이상 | 5,000억원 이상 |
| 거래량 | 일평균 10만주 이상 | 일평균 100만주 이상 |
| 재무 | 부채비율 200% 이하 | 영업이익 흑자 |

### 6.3 관리 규칙

| 상황 | 액션 |
|------|------|
| 매수 완료 | 관심종목에서 삭제 → 보유종목으로 이동 |
| 섹터 전망 하향 | 등급 하향 또는 삭제 |
| 3개월 이상 횡보 | 재평가 후 삭제 검토 |
| 실적 악화 | 즉시 삭제 |

## 7. 종목별 상세 전략 작성

### 7.1 필수 포함 요소

각 종목별 상세 전략에는 다음 요소를 반드시 포함합니다:

1. **기본 정보**: 매입가, 현재가, 수익률, 등급
2. **추세 분석**: 1개월 주가 변동, 52주 위치, 거래량 추이, 추세 점수
3. **섹터 전망**: 해당 섹터 전망 등급 및 영향 (신규)
4. **목표가/손절가**: 각각 산정 근거 포함
5. **액션 플랜**: 구체적인 매도 계획 (비율, 수량)
6. **종합 판단**: 별점(⭐) 또는 우선순위

### 7.2 작성 템플릿

```markdown
### X.X [등급] - [종목명]

| 구분 | 가격 | 산정 근거 |
|------|------|-----------|
| 매입가 | 00,000원 | - |
| 현재가 | 00,000원 | YYYY-MM-DD 종가 |
| 수익률 | ±00.00% | X등급 |
| **추세 점수** | **±N** | 1개월 ±X%, 52주 위치 X% |
| **섹터 전망** | **⭐⭐⭐⭐** | [섹터명] - 근거 요약 |
| **목표가** | **00,000원** | • 증권사 컨센서스<br>• 섹터 전망 반영 |
| **손절가** | **00,000원** | • 지지선 또는 -X% |

**액션 플랜:**
- **X원 도달**: Y주 매도 (Z%) → 예상 손익 ±N원
- **X원 하락**: 손절 실행

**종합 판단**: ⭐⭐⭐⭐ [간략 요약]
```


### 8.2 필수 섹션

1. 현황 분석 (등급별 분포, 섹터별 분석)
2. **섹터별 전망** (신규 - 보유 종목 관련 섹터 전망)
3. 리스크 등급 분류 (종목 리스트)
4. 등급 분류 근거 (A~F등급 각각)
5. 종목별 상세 전략 (목표가/손절가 포함)
6. 단계별 운영 전략 요약
7. 리스크 관리 원칙
8. 세금 최적화 전략
9. 사용자 견해 및 검토
10. 실행 체크리스트
11. 목표 설정
12. 관심종목 분석

## 9. 품질 관리

### 9.1 숫자 정확성

> ⚠️ **숫자 값의 정확성이 매우 중요**
> 
> - 모든 수치는 최신 데이터를 웹 검색으로 재확인
> - 신중하게 계산
> - **숫자 오류는 심각한 피해 초래**

### 9.2 검증 체크리스트

- [ ] 모든 종목 현재가 웹 검색으로 확인
- [ ] 평가손익 = (현재가 - 매입가) × 수량 검증
- [ ] 수익률 = (현재가 - 매입가) / 매입가 × 100 검증
- [ ] 등급 분류 기준 준수 확인
- [ ] 추세 점수 산정 근거 명시
- [ ] **섹터 전망 등급 및 근거 명시** (신규)
- [ ] 목표가/손절가 산정 근거 명시 (섹터 전망 반영)
- [ ] 관심종목 추세 분석 및 매수가 산정 확인
