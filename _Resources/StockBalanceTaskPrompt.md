# 주식 잔고 관리 작업 지시 프롬프트

## 파일 구조

| 시트 | 내용 |
|------|------|
| 주식잔고 | 보유 종목 현황 (종목코드, 종목명, 매입가, 보유수량, 현재가, 평가손익, 수익률 등) |
| 매매이력 | 거래 기록 (날짜, 종목명, 구분, 수량, 단가, 거래금액, 수수료, 실현손익, 수익률, 비고) |
| 손익요약 | 투자 손익 요약 + 월별 실현손익 집계 |

---

## 1. 매매이력 입력

### 기본 형식
```
[잔고 파일 업로드]

오늘(1/24) 매매이력 입력해줘:
- 삼성전자 10주 매수 85,000원
- 하나마이크론 100주 매도 26,500원 (일부 익절)
```

### 상세 형식 (매입가 지정 필요 시)
```
[잔고 파일 업로드]

오늘(1/24) 매매이력 입력해줘:

매수:
- 삼성전자 10주 85,000원 (신규 매수)

매도:
- 하나마이크론 100주 26,500원, 매입가 24,845원 (일부 익절)
```

### 당일 매수/매도 (단타)
```
오늘(1/24) 매매이력:
- 루미르 400주 매수 13,380원
- 루미르 400주 매도 15,360원 (단기 익절)
```

---

## 2. 종가 업데이트

### 직접 입력 (권장)
```
[잔고 파일 업로드]

종가 업데이트:
- 하나마이크론: 26500
- 코오롱인더: 48000
- 칩스앤미디어: 17800
- 다날: 8900
- 필옵틱스: 46050
...
```

### 자동 조회 요청
```
[잔고 파일 업로드]

보유 종목 종가 한 번에 조회해서 업데이트해줘
```

---

## 3. 월별 마감 처리

### 월말 마감
```
[잔고 파일 업로드]

1월 매매이력 마감 처리해줘:
- 1월 소계 행 추가
- 손익요약 시트 1월 데이터 확정
```

### 새 달 시작
```
[잔고 파일 업로드]

2월 첫 매매이력 입력해줘:
- 삼성전자 5주 매수 84,000원
```

---

## 4. 일일 리포트

### 기본 리포트
```
[잔고 파일 업로드]

오늘 일일 리포트 만들어줘
```

### 상세 리포트
```
[잔고 파일 업로드]

오늘(1/24) 일일 리포트:
- 총 평가금액, 전일 대비 변동
- 수익률 상위/하위 5개 종목
- 오늘 매매 내역 요약
- 실현손익 누계
```

---

## 5. 종합 요청 (일일 루틴)

### 장 마감 후 전체 처리
```
[잔고 파일 업로드]

오늘(1/24) 잔고 정리:

1. 매매이력
   - 하나마이크론 100주 매도 26,500원 (일부 익절)

2. 종가
   - 하나마이크론: 26500
   - 코오롱인더: 48000
   - (나머지 종목...)

3. 업데이트된 잔고 파일과 일일 리포트 만들어줘
```

---

## 6. 기타 요청

### 신규 종목 추가
```
신규 종목 추가:
- 종목코드: 005930
- 종목명: 삼성전자
- 매입가: 85,000
- 수량: 10
- 목표가: 95,000
```

### 종목 삭제 (전량 매도 후)
```
전량 매도 완료된 한국무브넥스 주식잔고에서 삭제해줘
```

### 손익 현황 확인
```
[잔고 파일 업로드]

현재 손익 현황 요약해줘
```

### 수익률 순위
```
[잔고 파일 업로드]

수익률 상위 5개, 하위 5개 종목 알려줘
```

### 특정 종목 매매이력 조회
```
[잔고 파일 업로드]

다날 매매이력 전체 보여줘
```

---

## 서식 규칙

### 숫자 표시
- 금액: 1,000단위 콤마
- 수익률: 퍼센트 (0.00%)
- 정렬: 숫자는 오른쪽 정렬

### 색상
- 양수 (이익): 빨간색
- 음수 (손실): 파란색

### 매매이력 시트 구조
```
1~3행: 안내 문구
6행: 기존 거래 (월초 누적)
7행: (빈 행)
8행: 헤더
9행~: 매매이력 (날짜순)
마지막: 월별 소계 → 실현손익 합계
```
